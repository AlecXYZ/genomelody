Upload File

To get started, upload a DNA sequence in FASTA format. This file contains the nucleotide data that will be transformed into music. After uploading your file, select the instrument you'd like to use for playback. It will determine the tone and character of the resulting sound.

Now, define the range of lines from the DNA sequence you want to convert. Remember, the more lines you include, the longer the conversion process will take and the longer the audio file will be.

Set Up Conversion

Choose an algorithm to determine the note assignment and set the tempo for playback speed.

Algorithm Description

The algorithm parses lines from a file containing nucleotide sequences into pairs. Each individual nucleotide, as well as nucleotide pairs composed of a single repeated nucleotide (except TT), is assigned a note from the diatonic scale, which consists of seven notes.
